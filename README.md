# Lex

Документы и чек-листы.

[![.NET](https://img.shields.io/badge/.NET-9.0-purple)](https://dotnet.microsoft.com/)
[![MudBlazor](https://img.shields.io/badge/MudBlazor-v8.15.0-blue)](https://mudblazor.com/)

## Возможности

- Каталог шаблонов документов
- Интерактивные чек-листы
- Черновики и версионирование

## Стек технологий

- .NET 9
- MudBlazor
- SQLite + EF Core
- ASP.NET Core Identity

## Быстрый старт

```bash
git clone https://github.com/just-between-us/LexFlow.git
cd LexFlow
dotnet restore
dotnet ef database update
dotnet run --project Lex
```
